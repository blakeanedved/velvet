# velvet

A access control/authorization library inspired by the prolog language and [oso], designed with an async-first mindset. 

[oso]: https://crates.io/crates/oso

## Installing

Make sure to add `velvet` to your `Cargo.toml`, like:

```toml
[dependencies]
velvet = "0.0.1"
```
